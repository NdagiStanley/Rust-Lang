# Rust

The Rust Programming Language

## Installation

Run:

```bash
brew install rust
```

Or the recommended way from rust's website:

```bash
curl https://sh.rustup.rs -sSf | sh
```

## Implementation

Run:

```bash
rustc hello-world.rs
```

then:

```bash
./hello-world
```
